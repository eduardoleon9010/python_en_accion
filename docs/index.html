🎓 Introducción

Este repositorio está dedicado a facilitar el aprendizaje de Python con fines académicos. Contiene scripts explicativos, ejercicios de práctica y proyectos reales para estudiantes, docentes e investigadores.

📁 Estructura del repositorio

  • scripts/: Programas organizados por nivel de dificultad.
  • ejercicios/: Problemas prácticos para desarrollar habilidades.
  • proyectos/: Casos de aplicación con propósitos científicos o educativos.
  • documentacion/: Explicaciones detalladas, guías y referencias.
  • recursos/: Datasets, presentaciones y materiales de apoyo.
