Acerca del Proyecto

Este repositorio está diseñado para facilitar el aprendizaje progresivo del lenguaje Python mediante ejemplos prácticos, ejercicios de desarrollo y proyectos reales en áreas como análisis de datos, visualización, simulación y desarrollo de juegos.

Estructura del repositorio

  • scripts/ – Programas organizados por nivel (básico a avanzado)
  • ejercicios/ – Problemas prácticos para fortalecer habilidades
  • proyectos/ – Aplicaciones completas (juegos, análisis de datos, etc.)
  • documentacion/ – Guías y explicaciones adicionales
  • img/ – Imágenes de apoyo para notebooks y presentación
  • docs/ – Esta página web interactiva (GitHub Pages)
